# CC Sama Railway Fixed

Projeto pronto para subir no Railway.

Arquivos importantes:
- `server.js`: backend Express + SQLite + serve todos os arquivos de `public/`.
- `public/index.html`: app mobile/editor v3.
- `public/cc_sama_bolso_sql_v3_editor.html`: mesma versão v3.
- `public/cc_sama_db_inspector.html`: página para checar o que está salvo no banco.

Comandos locais:

```bash
npm install
npm start
```

URLs locais:
- http://localhost:3000/
- http://localhost:3000/cc_sama_db_inspector.html
- http://localhost:3000/api/status

No Railway, suba a pasta inteira no GitHub e faça Deploy from GitHub.
