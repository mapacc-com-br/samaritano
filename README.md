# Mapa de Cirurgias - Escala por Dia

Atualização:

- Anestesista agora tem:
  - horário de escala
  - função
  - observação
- A escala é vinculada à data selecionada.
- Um anestesista cadastrado em 15/05/2026 não aparece automaticamente em 16/05/2026.
- Se o mesmo anestesista for cadastrado novamente no mesmo dia, atualiza os dados em vez de duplicar.
